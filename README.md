## Project Overview
This project analyzes website traffic data and uses Machine Learning to predict traffic sources based on user behavior metrics. A Random Forest Classifier is used to classify traffic sources and identify the most important factors influencing website visits.

## Objectives
- Analyze website traffic patterns.
- Handle missing values in the dataset.
- Encode categorical traffic source data.
- Train a Random Forest Classification model.
- Evaluate model performance using accuracy and classification metrics.
- Identify important website engagement factors.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Dataset
The project uses a website traffic dataset (`website_traffic.csv`) containing:
- Traffic Source
- Page Views
- Session Duration
- Bounce Rate
- Time on Page
- Previous Visits
- Conversion Rate
